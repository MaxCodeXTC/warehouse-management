Overview

The primary target of the system is to help in making predictions for the perishable goods in a general warehouse. The two possiblities which are major source of problems,

  • Understocking

  • Overstocking

The above two possiblities can be managed with utmost supervision of demand and order.



Focus Area

Demand:

  • The project main focus is on demand prediction for perishable goods.

  • The model used to predict perishable goods demand is CNNLSTM (Convolution Neural Network with Long Short Term Memory).


Order:

  • The predicted demand is then used to predict the order quantity and order cycle.

  • The model used to predict perishable goods order quantity and cycle is EOQ (Economic Order Quantity).



Analysis

The project also involves visual analysis of the predicted items.

  • Predicted Demand - N Days graph

  • Predicted Order quantity - N Days graph

  • Predicted Order Cycle - N Days graph

Where N is the number of days on which prediction has been performed.
